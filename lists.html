3. Descriptive List

Here is a list of definitons of computer science terms.

Number:
Represents numeric values, including integers and floating-point numbers. For example: 42, 3.14.
String:
Represents sequences of characters, such as text. For example: "Hello, World!", 'JavaScript'.
Boolean:
Represents a binary value, either true or false. Used for logical operations and comparisons.
Undefined:
Represents a variable that has been declared but has not been assigned a value yet. Variables are initialized with undefined by default.
Null:
Represents the intentional absence of any object or value. It is often used to signify that a variable should have no value.
BigInt:
Also introduced in ECMAScript 11 (ES11), BigInt is used for working with arbitrarily large integers. It is represented by adding an n to the end of a numeric literal, like 10n or 1000000000000000000000n.
Symbol:
Introduced in ECMAScript 6 (ES6), symbols are unique and immutable values that can be used as object property keys. They are often used for creating private object properties.
